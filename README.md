基于Koa2的API框架
====================

框架目录：
~~~
|-- config            配置文件目录
|-- controller        控制器
|-- libs              框架核心类库
|   | -- decorator    装饰器
|   | -- exception    错误类型
|   | -- middleware   中间件
|   | -- token        Token令牌
|   | -- validate     验证器
|-- model             模型
|-- router            路由
|-- utils             工具  
~~~

## 优点：
1. AOP：Exception、Validate、Auth
2. MVC：router、controller、model、service
3. Auth：Token
4. Functional programming
5. ES6：decorator, async function, etc.
6. ORM：bookshelf.js
